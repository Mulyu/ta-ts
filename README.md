# ta4ts

Technical Analysis for Typescript.
ta4ts is not depends on [TA-Lib](https://ta-lib.org/), so it run on any Javascript runtime.(like browser)

# Features

ta4ts supports indicators as below.

- [x] sma
- [x] ema
- [x] macd
- [ ] wma
- [ ] mma
- [ ] hma
- [ ] kama
- [ ] lwma
- [ ] double ema
- [ ] rsi
- [ ] stochastic
- [ ] parabolic sar
- [ ] williams r
- [ ] bollinger band

# Install

```bash
npm install ta4ts
```
